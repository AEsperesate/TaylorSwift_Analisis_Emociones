# Análisis emocional de la discografía de Taylor Swift

Este proyecto analiza las emociones presentes en la discografía de **Taylor Swift** utilizando Python y R.  
Se estudian los álbumes, las canciones y la influencia de los distintos colaboradores y de la duración de las canciones.

## Contenido del repositorio

- `src/` – Código fuente en Python y R para procesar letras y generar gráficos.  
- `data/csv/` – Metadatos de las canciones y álbumes (datos obtenidos de Wikipedia).  
- `data/processed/results/` – Resultados del análisis de emociones (CSV).  
- `insights/` – Informe en PDF con insights y visualizaciones detalladas.  

## Cómo explorar el proyecto

1. Consultar directamente el PDF del informe en `insights/` para ver resultados y gráficos.  
2. Abrir `src/lyrics_analysis.py` y `src/emotions.R` si quieres reproducir el análisis o generar nuevos gráficos.

## Tecnologías

- Python (pandas, transformers)  
- R (tidyverse, ggradar, scales, forcats)  

## Nota

No se incluyen las letras originales de las canciones por derechos de autor.
